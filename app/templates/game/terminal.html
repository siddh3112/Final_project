> SECTOR: DATA_CLASSIFICATION

              
> DATA comes in three forms. STRUCTURED fits neat rows and columns — a spreadsheet of names, ages and balances, easily queried with SQL. UNSTRUCTURED has no schema and is stored raw: emails, images, audio, chat logs. SEMI-STRUCTURED sits between the two — tags or keys give partial order without a rigid table (JSON, XML).
> An estimated 80–90% of enterprise data is unstructured, yet most legacy tools were built for the structured ~10–20% — so dashboards often show only a thin slice. Watch for DARK DATA too: information collected but never used (it can even be tidy, structured records).
> Drag each item on the conveyor into the correct bin. This is a PRACTICE run — a warm-up for the graded classification that follows the reboot.
